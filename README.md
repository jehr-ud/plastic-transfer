# Plastic Transfer

Framework for skill-based transfer learning in reinforcement learning.

## Features

- Dynamic skill creation (PPO-based)
- Temporal representation of trajectories
- Structural similarity matching
- Reusable adaptation

## Usage

```python
from plastic_transfer.core.plastic_transfer import PlasticTransfer
```
